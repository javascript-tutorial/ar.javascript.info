# قم بتحويل 'if..else' إلى '?'

قم بإعادة كتابة `if..else` باستخدام العديد من معامل الشرط `'?'`.

ينصح بتقسيم الكود إلى أكثر من سطر ليكون مقروء.

```js
let message;

if (login == 'Employee') {
  message = 'مرحبًا';
} else if (login == 'Director') {
  message = 'تحياتي';
} else if (login == '') {
  message = 'لا دخول';
} else {
  message = '';
}
```
